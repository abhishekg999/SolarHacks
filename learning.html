How do I code?

Here is a short page of simple code commands in Python

Printing Commands

Use print() to print commands
For example:
Input: print("Hello World!")
Output: Hello World!

Declaring Variables

Use = to set a value to a variable
x = 100 //assigns the number 100 to the variable x
word = "Hello" //assigns the string "Hello" to the variable word

Variable Operations

Math operations follow PEMDAS, they can be in ()
+ : Addition
- : Subtraction
* : Multiplication
/ : Division
% : Modulus
Examples:
x = 5 + 8 //assigns the value 13 to the variable x
y = 44 - 2 - 2 //assigns the value 40 to the variable y
y = 44 - 2 - 2 //assigns the value 40 to the variable y
z = 2*5 //assigns the value 10 to the variable z
a = 22%10 //modulus is remainder; assignes the value 2 to the variable a
newword = "Hello" + "_World" //new string Hello_World is assigned to the variable newword

Arrays

Arrays let you store multiple values in one variable
a = [1, 2, 3, 4, 5] //the array [1, 2, 3, 4, 5] is stored to the variable a
print(a[0]) //Access array with this notation. Arrays start with 0. This prints the number 1
a[2] = 44 //Changes the value of a[2] to 44. The new array is[1, 2, 44, 4, 5]

Loops

For loops let you repeat a task multiple types
for q in range(len(a)):
a[q]=80
//The variable q changes from 0 to the length of the array a. This changes the array to [80, 80, 80, 80, 80]
